A Bit About Me

Born in Chicago and raised in Ohio, I am a Midwesterner at heart, with a New York City exterior. After graduating with honors from St. Vincent-St. Mary High School in Akron, I enrolled at New York University where I studied filmmaking, ultimately working in the film and television industry for more than ten years.

About a year ago I became interested in learning to code. I did my research, found some online tutorials and began learning. Almost immediately I recalled programming games when I was a little girl, painstakingly typing code line by line into my Texas Instruments TI 99/4A computer.

After studying on my own for a few months, I enrolled in Project Code, sponsored by the New York Public Library, to study HTML and CSS earlier this year. I absolutely loved it! Shortly thereafter I relocated back to Ohio and enrolled at We Can Code IT here in Cleveland. I successfully completed a 12-week full-stack development bootcamp focused on C#/.NET programming.

Skills: C# • .NET • Active Server Pages / ASP.net • JavaScript • JQuery • Bootstrap • MVC • HTML5 • CSS3 • Object Oriented Programming (OOP) • Responsive Design / Mobile • Structured Query Language (SQL) • Transactional Databases / MS SQL • Source Control / GitHub
